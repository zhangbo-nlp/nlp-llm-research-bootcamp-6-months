## 信息获取习惯

建议学生每周固定浏览：

- arXiv cs.CL：https://arxiv.org/list/cs.CL/recent
- Papers with Code NLP tasks：https://paperswithcode.com/area/natural-language-processing
- Hugging Face Daily Papers：https://huggingface.co/papers
- ACL Anthology：https://aclanthology.org/

阅读新论文时不要追求“看懂每个公式”，先回答四个问题：它解决什么问题、用了什么方法、和什么基线比、结论是否可信。
